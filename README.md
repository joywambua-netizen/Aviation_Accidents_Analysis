# Aviation Accident Data Analysis

## Project Overview
This project performs an Exploratory Data Analysis (EDA) on aviation accident data to identify safety trends across various aircraft makes and sizes.

## Methodology
- **Data Cleaning:** Standardized column names and handled missing values.
- **Metric Creation:** Developed an `Injury_Fraction` metric to normalize injury severity.
- **Statistical Filtering:** Limited analysis to manufacturers with at least 10 recorded incidents to ensure statistical significance.

## Key Findings
- **Small Aircraft:** Exhibit higher volatility in safety outcomes, suggesting that operational factors significantly impact results.
- **Large Aircraft:** Show more consistent safety outcomes, indicating high adherence to standardized commercial safety protocols.

## Recommendations
- Based on the analysis, we recommend benchmarking safety protocols against top-performing manufacturers identified in this study.
